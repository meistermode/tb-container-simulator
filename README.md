# просто симулятор открытия мистиков (не 1:1)

ну чот мне было скучно и я сделал ну вот такое вот

## возможности

- магаз с контейнерами (сам обновляться должен ну наверн не знаю как получится)
- портирована система с уровнями, с талисманами (гаранты) и всё такое
- хранилище есть, где ресурсы все можно выбитые посмотреть
- на старте даётся 50к голды, а дальше сами

## установка

можете .exe файл запустить готовый (он полностью чист), а можете собрать свой из исходников (интернет в помощь)

## примечание

если там баги какие-то будут, или что-то просто будет плохо работать (не как в игре), то я ничо не буду чинить, лень
